This FHIR Implementation Guide (IG) was created as a demonstrator artefact that was informed by the Botswana training use case as defined below.

The Botswana FHIR capacity building & training took place from 18-Sept-2023 to 22-Sep-2023.

This documentation and set of artefacts are still undergoing development. This content is only for demonstrative purposes.
